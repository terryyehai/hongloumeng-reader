# 紅樓夢第一百回

## 薛文龍悔娶河東獅 賈迎春誤嫁中山狼

話說薛蟠從江南回來了，一到家便去見了母親和妹妹。薛姨媽見兒子平安回來，自然歡喜。寶釵便問道：「哥哥這趟江南之行，可順利麼？」薛蟠道：「還算順利，只是路上遇見幾個朋友，喝了幾場酒耽擱了些日子。」薛姨媽道：「回來就好，以後少在外面閒逛。」薛蟠答應著，便回自己房裡去了。

這薛蟠自從娶了夏金桂為妻，家中便不得安寧。那夏金桂本是個潑辣貨色，嫁過來後仗著自己有幾分姿色，便在家中橫行霸道。先是排擠香菱，後來又處處與薛蟠作對。薛蟠本是個軟弱之人，被她一鬧，便漸漸怕了她。

這一日，夏金桂對薛蟠道：「你終日在外頭跑，也不管家裡的事。我這裡還有個丫頭叫寶蟬的，甚是調皮，你若要了她，我也省心。」薛蟠本就好色，聽說便答應了。誰知這是夏金桂的計策，她是想借刀殺人，讓寶蟬去對付香菱。

那寶蟬本是夏金桂的丫鬟，生得有些妖艷，心裡也不安分。得了這個差使，便去找香菱的麻煩。香菱本是個老實人，哪裡經得起這般折騰？便被逼得病的病，瘦的瘦。

薛姨媽看在眼裡，心疼香菱，便對薛蟠道：「你這媳婦也太不像話了，香菱招她惹她？這樣折磨人家。」薛蟠道：「母親別管她，她就是這脾氣。」薛姨媽道：「都是你慣的！」

這邊薛家的事情暫且不提，單說賈迎春那邊。

賈迎春自從嫁給孫紹祖後，便沒有過過一天好日子。那孫紹祖本來就是個無賴，當初求親時花言巧語，娶過門後便露出本色。動不動就打罵迎春，把她當丫頭使喚。

這一日，孫紹祖從外面喝得醉醺醺的回來，見迎春便罵道：「你這賈家的丫頭，別以為你家老子做過官便了不起。如今你老子死了，你還不是落到我手裡？」迎春低頭不敢則聲。

孫紹祖又道：「當初你家陪嫁的那些東西，早被我輸得差不多了。你還不快去跟你們賈家要？」迎春道：「婆婆已經去世了，家裡的事我也做不得主。」孫紹祖聽了便是一記耳光，打得迎春嘴角出血。

迎春心中悲痛，卻又無處可去。只得在房中垂淚，想著在賈府時的日子，那時姊妹們一起吟詩作對，何等快活。如今落到這個田地，真是生不如死。

這事傳到賈府眾人耳中，皆是不平。賈探春道：「二姐姐也太軟弱了，若是換了我，早跟那畜生拼了。」李紈道：「她也是沒法子的，嫁出去的女兒潑出去的水，我們就是想管也管不著。」王熙鳳道：「改日我去看看她，好歹她是咱們家的人，不能讓人欺負狠了。」

卻說這一日，賈政接到一封信，說是賈雨村那邊有公事要商量。賈政看了信，便對王夫人道：「雨村老爺約我明日過府一敘，想是為了那甄家的事。」王夫人道：「老爺自去吧，家裡有我。」

賈政第二日便去了賈雨村那裡。兩人說了一回話，賈雨村道：「老大人，如今甄家雖然抄沒，但甄老爺的餘款還有一些。只是這裡頭有些麻煩，需得老大人從中疏通。」賈政道：「這是如何說起？」賈雨村道：「甄家還有些田莊在外面，只是如今被人佔了。若是大人肯出力，將來的好處自然少不了的。」

賈政聽了，心中暗暗吃驚，想不到甄家還有這些首尾。當下也不便說什麼，只說「容我想想」便告辭回來。

回到家後，賈政想著這件事，左右為難。若是幫了賈雨村，於心不安；若是不幫，又怕得罪了他。想了半日，終究沒有主意。

這正是：
```
甄家有寶藏機關，雨村從中弄機巧。
賈政為難難自決，忠奸不辨太糊塗。
```

欲知後事如何，且看下回分解。

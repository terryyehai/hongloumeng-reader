# 紅樓夢第一百九回
## 記微嫌娟女用奇謀　賈元春托夢警痴人

話說賈母自王夫人回去之後，連日身上不好，見了姊妹等，只是落淚。賈政日夜在家陪侍，又見外面沒有甚麼事，逐日到家安慰說：「老太太這病若好了，少不得論理理家政，現在外面買賣倒還可以支撐。」賈母道：「我老了，骨頭裡的血都冷了，生不得氣了。人家說我固執，我也只好看著罷。你且料理你的正經事去，不要只管來歪纏我。」賈政不敢再說，只得住下。

那時正值臘月，天氣寒冷。一日史侯家打發人來送上年菜。賈母便留住吃了晚飯再去。那人回來說：「我們老爺、太太說，年下事情多，沒甚麼孝敬老太太的，只有一點心意，請老太太賞收罷。」賈母道：「難為你們老爺、太太費心。只是我們這裡也不短甚麼，你們留著自己用罷。」又賞了那人幾百錢。

那人一去，賈母便覺得身上乏了，便歪在榻上。只見鴛鴦走來道：「老太太，用熱水泡泡腳罷？」賈母道：「不用了，你且去歇著。」鴛鴦答應下去。不一時，琥珀走來說：「太太那邊打發人來說，王夫人請老太太說話。」賈母道：「這會子天晚了，有甚麼話明日再說罷。」琥珀道：「來人說有要緊的事。」

賈母只得起來，同琥珀到王夫人房中。只見王夫人與妹妹薛姨媽在那裡說話，似有甚麼機密之事。賈母問道：「甚麼事這般鄭重？」王夫人道：「老太太坐下，我有話要說。」原來這件事與賈環有關。

卻說賈環自從抄檢大觀園之後，在府裡頗不得意。他素日嫉妒寶玉，今見寶玉病倒，便在背後對人說：「寶玉不知會不會好，若是好了，也是個廢物。」這些話傳到趙姨娘耳中，趙姨娘便起了心思。

原來趙姨娘素日與馬道婆來往密切。那馬道婆會些旁門左道，專門幫人暗中害人。這日趙姨娘便請了馬道婆來，兩人密商許久。馬道婆道：「要破壞寶玉不難，只是需要個引子。」趙姨娘道：「你只管說要甚麼。」馬道婆道：「需要一件貼身的東西，上面有寶玉的氣息。」

趙姨娘回去，便向賈環要了寶玉平時用的手帕子。賈環問道：「你要這個做甚麼？」趙姨娘道：「你別管，我有用處。」賈環便把平日從寶玉那裡得來的手帕给了趙姨娘。趙姨娘得了手帕，便去找馬道婆。

馬道婆接了手帕，便做起法來。她先用符紙燒了，又把一些粉末灑在手帕上，口中念念有詞。說也奇怪，那手帕便漸漸發出一般異香。馬道婆道：「成了。只要把這帕子放在寶玉枕頭底下，便可叫他不知不覺中邪。」

趙姨娘得計，便乘人不備，把這帕子放在寶玉枕頭底下。說來也怪，寶玉自從枕了這枕頭，便昏昏沉沉，儘說胡話。一會兒說看見甚麼鬼，一會兒又說看見和尚道士來度他。賈政請了太醫來看，太醫看了說是邪火入體，開了方子吃了也不見效。

這日寶玉正在床上躺著，只見林黛玉走來道：「寶玉，你好了麼？」寶玉道：「林妹妹，我正找你呢。我有話對你說。」黛玉道：「甚麼話？」寶玉道：「我的心裡話，只是說不出來。」說著便哭了。黛玉道：「你別哭，我曉得你的心。」正說著，只見紫鵑走來道：「姑娘，老太太那邊有事請你過去。」黛玉只得去了。

這裡寶玉又昏睡過去。夢中只見一位仙子走來道：「我是警幻仙姑座下的仙女，特來點化你。你根基原是不俗的，只因迷於富貴聲色，漸漸墮落。如今若不回頭，後來悔之無及。我今給你一卷《石頭記》，你自己參詳罷。」說罷遞過一本書來。寶玉接過一看，只見封面寫著《石頭記》三字，裡面密密麻麻都是字，卻看不大懂。正想問時，那仙女道：「你且回去，日後自然明白。」說罷飄然而去。

寶玉醒來，卻把夢中之事忘了一半。只記得那仙女說的話語些，因對襲人道：「你去回了老太太，就說我好了，明日要去給老太太請安。」襲人道：「好了麼？還是再請太醫來看看罷。」寶玉道：「不用了，我已經好了。」

這話傳到賈母那裡，賈母便來看寶玉。見寶玉果然精神了些，不似先前那樣糊塗。賈母道：「阿彌陀佛，總算好了。這孩子從小就愛說些奇怪的話，如今大了倒好了些。」又對王夫人道：「你們好生照顧他，不可再讓他接触那些不乾淨的東西。」王夫人答應著。

賈母又問起府中各項事務，王夫人一一回了。賈母道：「如今家裡用度大，不如從前。你們也要節省些，不要只顧著排場。」王夫人道：「老太太說的是，我們理会得。」

說話間，只聽外面有人回說：「甄家來了人，在外面求見。」賈政道：「甄家？哪位甄家？」家人道：「就是金陵的甄家，打發人來請安。」賈政道：「請進來罷。」不一時，甄家來的人進來行了禮，說道：「我們老爺問各位老爺、太太、哥兒、姐兒安好。我們老爺說，多年不見，很是想念，特打發我們來請安。」賈政道：「難為你們老爺費心。你回去說，我們都好，請他不必掛念。」

甄家人又道：「我們老爺說，最近家裡出了些事，想來問問大家的安。」賈政道：「出了甚麼事？」甄家人道：「說是家裡有人作祟，鬧得不安寧。」賈政聽了暗想道：「甄家也是大户人家，竟也出了這些事。」便道：「原來如此。你回去說，我們知道了，請他放心。」甄家人答應去了。

這裡賈母問道：「甄家出了甚麼事？」賈政道：「說是家裡有人作祟。」賈母道：「也難怪，如今這府裡也不是從前，亂得很。」說著又咳嗽起來。鴛鴦忙上前捶背。

早有李紈、鳳姐兒等聞知，都來請安。鳳姐兒道：「老太太好點了麼？」賈母道：「老了，不中用了。你們各自忙去罷，不用都在這裡。」鳳姐兒答應著，便與李紈退出。

走到園中，李紈道：「寶玉這病倒也奇怪，好端端的就中了邪。」鳳姐兒道：「誰說不是呢。也許是年輕人陽氣弱，碰見了甚麼不乾淨的。」兩人說著，各自回房去了。

卻說那趙姨娘自從做了那件事之後，心裡也有些不安。這日正在房中，只見馬道婆走來道：「姨娘放心，那件事成了。只是事成之後，我要謝禮。」趙姨娘道：「你要多少？」馬道婆道：「五十兩銀子。」趙姨娘道：「這會子一時拿不出，改日罷。」馬道婆道：「姨娘別糊弄我，我曉得你有的。」趙姨娘無奈只得答應。

那馬道婆得了銀子，便離開了。她徑自走到一所破廟裡，把那手帕燒了，口中說道：「怨不得我心狠，只怪你們太狠毒。」說罷揚長而去。

不表馬道婆單說賈府中人。那賈環自從得了那帕子與趙姨娘之後，心裡也有些犯疑。這日正在那裡胡思亂想，只聽見外面有人說：「二爺，老太太叫你去呢。」賈環只得去見賈母。

賈母見了賈環道：「環兒，你近來在學裡功課如何？」賈環道：「還好。」賈母道：「你也要爭氣才是。你三妹妹天天用功，你也要學學她。」賈環答應著，心裡卻不高興。賈母又道：「你去罷，好生讀書。」賈環答應退出。

賈環走在路上，心裡想道：「老太太只疼愛寶玉，那里把我放在眼裡。總有一天，我要叫他們好看。」正想著，只見迎春走來道：「二弟弟去哪裡？」賈環道：「回房去。」迎春道：「你近來可好？」賈環道：「好甚麼，天天挨數落。」迎春道：「你也別多想，好好讀書總會有用的。」賈環道：「罷了罷了，你們都會做好人，只有我壞。」說罷便走了。

迎春看著他的背影，搖頭道：「這孩子也是可憐。」便各自去了。

單說賈政那日送走甄家之人，便對賈赦道：「甄家來人說家裡不安寧，想來也是那些爭端。」賈赦道：「甄家老爺也是糊塗，家裡事都管不住。」賈政道：「我們家又何嘗不是如此。」賈赦道：「我說你也別管那麼多，各自過各自的日子罷。」

賈政嘆道：「話雖如此，只是 parent children，不得不管。」兩人說了一回，各自安歇。

如今且說寶玉自從好了之後，倒也安分了些。這日對襲人道：「我前日做了一個夢，夢見一位仙女，給了我一本《石頭記》。」襲人道：「甚麼《石頭記》？」寶玉道：「我也看不大懂，只是上面的故事倒有趣。」襲人道：「二爺別胡思亂想了，好生將養著罷。」寶玉答應著。

只是那《石頭記》上的故事，寶玉後來倒記得了些。只是後事如何，且聽下回分解。

賈母見了賈環道：「環兒，你近來在學裡功課如何？」賈環道：「還好。」賈母道：「你也要爭氣才是。你三妹妹天天用功，你也要學學她。」賈環答應著，心裡卻不高興。賈母又道：「你去罷，好生讀書。」賈環答應退出。

賈環走在路上，心裡想道：「老太太只疼愛寶玉，那里把我放在眼裡。總有一天，我要叫他們好看。」正想著，只見迎春走來道：「二弟弟去哪裡？」賈環道：「回房去。」迎春道：「你近來可好？」賈環道：「好甚麼，天天挨數落。」迎春道：「你也別多想，好好讀書總會有用的。」賈環道：「罷了罷了，你們都會做好人，只有我壞。」說罷便走了。

迎春看著他的背影，搖頭道：「這孩子也是可憐。」便各自去了。

單說賈政那日送走甄家之人，便對賈赦道：「甄家來人說家裡不安寧，想來也是那些爭端。」賈赦道：「甄家老爺也是糊塗，家裡事都管不住。」賈政道：「我們家又何嘗不是如此。」賈赦道：「我說你也別管那麼多，各自過各自的日子罷。」

賈政嘆道：「話雖如此，只是 parent children，不得不管。」兩人說了一回，各自安歇。

如今且說寶玉自從好了之後，倒也安分了些。這日對襲人道：「我前日做了一個夢，夢見一位仙女，給了我一本《石頭記》。」襲人道：「甚麼《石頭記》？」寶玉道：「我也看不大懂，只是上面的故事倒有趣。」襲人道：「二爺別胡思亂想了，好生將養著罷。」寶玉答應著。

只是那《石頭記》上的故事，寶玉後來倒記得了些。只是後事如何，且聽下回分解。

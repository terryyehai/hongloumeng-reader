# 第一百零六回　賈母祈福消災難　鳳姐煎心淚暗流

話說賈府被抄之後，眾人皆是驚慌失措。那賈母到底是經過大事的人，心裡雖然著急，面上卻還能沉得住氣。

這日，只見賈母叫過鴛鴦來，說道：「你去告訴太太、奶奶們，從今日起，各房裡都要省儉些。咱們家如今不比從前，不可再大手大腳了。」

鴛鴦答應著去了。

不多時，只見王夫人、邢夫人、李紈、鳳姐等都來了。

賈母說道：「你們也都知道，如今咱們家遭了難，外面的人不知道有多少在笑話咱們。我這把老骨頭，倒還撐得住，只是苦了你們這些年輕的。」

眾人聽了，都低頭不語。

鳳姐在旁說道：「老太太放心，咱們總會有東山再起的一天。」

賈母搖頭道：「東山再起？說得倒容易。如今咱們家裡的銀子、田地都被抄了去，剩下的也不多了。以後的日子，可怎麼過？」

說著，便落下淚來。

眾人連忙勸道：「老太太莫要傷心，身子要緊。」

賈母道：「我如何不傷心？當年老爺在世的時候，咱們家何等風光！如今——罷了，不說也罷。」

眾人都不敢接口。

過了一會，賈母又說道：「我今日叫你們來，也不為別的。只是一件，我想著咱們家如今這個樣子，不如去寺裡走走，求求佛祖保佑。或者能有些轉機，也未可知。」

眾人都說：「老太太說的是。」

當下，便決定了明日去鐵檻寺燒香祈福。

次日，只見賈母換了新衣服，帶著王夫人、鳳姐、李紈等人，乘坐轎車，往鐵檻寺而去。

那鐵檻寺的和尚們聽說賈府太夫人來了，忙不迭地出來迎接。

賈母下了轎，只見那寺門口已經有不少善男信女在燒香禮佛。賈母也不多言，徑直走進殿去。

那殿中供的是觀世音菩薩，只見那菩薩寶相莊嚴，端坐在蓮花座上。賈母看了，不覺心中起敬，便跪了下來，磕了頭，口中念念有詞，也不知說些什麼。

眾人也都跟著磕頭。

磕完了頭，賈母便叫小和尚拿了香火錢來賞他。那小和尚連忙道謝不迭。

出了大殿，賈母又往後面走去。只見那後院中有幾株老松，蒼勁挺拔，倒也有幾分景致。

賈母在院中走了走，便對眾人說道：「這寺裡倒還清靜，比咱們家裡那些勾心鬥角的地方，好得多。」

鳳姐道：「老太太若喜歡，以後常來走走也好。」

賈母道：「也只能如此了。」

當下，便打道回府。

回到家中，賈母只覺得累了，便先去歇息。

卻說那鳳姐回到房中，想著今日之事，不覺傷心。她想道：「從前我在家裡，何等的风光！如今——咳，真是世事無常。」

那平兒在旁勸道：「奶奶莫要太傷心了身子要緊。」

鳳姐道：「你不懂。我如今這個樣子，在家裡還有什麼面子？那些人表面不說，心裡不定怎麼笑話我呢。」

平兒道：「奶奶別多想，自己身子要緊。」

鳳姐道：「我如何不想？從前老太太偏疼我，如今——罷了，不說了。」

說著，便躺了下來。

過了幾日，賈母的身子有些不爽，便叫請了大夫來看。

那大夫診了脈，說道：「老太太這是心火太旺，需得靜養。我開個方子，你們照著抓藥罷。」

說完，便開了方子去了。

賈母吃了藥，倒也好了些。

這日，賈母把眾人都叫到跟前，說道：「我這病好了也算是僥倖。以後你們都要各自珍重，莫要再像從前那樣不懂事了。」

眾人都答應著。

賈母又道：「我如今也不求別的，只求咱們家裡平安和睦，便是最大的福分了。」

說完，便叫眾人散了。

不知後事如何，且聽下回分解。
